# Minicode

## Архитектура

### App

Управляет взаимодействием с пользователем и CLI.

### Core

Содержит главный функционал приложения.

1. Config загружает .minicode.json отовсюду (AWD, CWD, UWD)
    1. Предоставляет апи для листинга моделей и агентов
    2. Предоставляет апи для получения конфига модели / агента
2. Agents позволяет создавать агентов (принимает модель и конфиг)
    1. Агент загружает плагины (вызывает setup у проинициализированных плагинов, если есть это в конфиге)
    2. Агент загружает тулзы в тулбокс (у каждого агента свой тулбокс) - для Plan будут только write_file
    3. Агент предзагружает скиллы
    4. Агент принимает первичный промпт
3. Session позволяет создавать сессии, принимает агента (в нём уже тулзы и готовые к загрузке скиллы)
    1. Позволяет форкать сессии
4. Шлюз для плагинов
    1. Собирает (регистрирует) все плагины по конфигу
    2. Для каждого агента создается поле plugins которое вызывает нужный плагин
    3. При инициализации агента он через шлюз врубает плагины
    4. Также могут быть плагины для сессии (например подсчет статы)
    5. Также могут быть плагины для приложения (app level)

## Plugins

1. Files позволяет работать с файлами - добавляет тулбокс files, подкгружает тулзы, включен по умолчанию в ., но можно переопределить в --file-root + добавляет @ выражения @file, @dir
2. Skills подгружает скиллы, добавляет тулбокс, загружает все видимые скиллы в контекст (только мету), добавляет @skill
3. Configurator добавляет скиллы и мб тулзы чтобы работать с конфигом .minicode.json (может регистрировать модели и создавать агентов)
4. Stats ловит инфу о сессии, хранит её, рисует после каждого хода (ход = несколько шагов?) + трекает стоимость
5. Compact добавляет команды /strip (удалить ответы тулзов), /slide (сократить контекст до последних Н сообщений), /compact (по умному сокращает контекст + можно докинуть промпт) + поддерживает пины?
6. MCP добавляет поддержку mcp серверов из конфига + можно выключать включать внутри по команде /mcp - можно будет проверить на пупетире и фигме
7. AGENTSMD добавляет автозагрузку agents.md или claude.md из рута (нужен файловый плагин чтобы узнать откуда что брать)
8. Delegator - добавляет тулбокс для делегации, а также принудительно загружает скилл для правильно работы с делегацией - только синк режим
9. Todo - добавляет тулбокс для тудушек, позволяет пользаку обновлять его руками, рисует его после обновления
10. Skills discovery - автоматически ищет скиллы opencode / claude и подгружает их мету в контекст
