# minicode

Упрощенный, CLI-friendly AI агент для программирования, вдохновленный OpenCode. Создан для разработчиков, которым нужен минималистичный опыт с тонкой настройкой.

## Обзор

minicode — это игрушечный проект, исследующий AI-ассистента для программирования с акцентом на простоту и кастомизацию. Предоставляет интерактивный REPL интерфейс для диалогового кодирования с поддержкой выполнения инструментов и выбора модели.

## Возможности

### Базовый функционал

- **Sticky REPL с поддержкой escape** - Интерактивный интерфейс командной строки с корректной обработкой ввода
- **Основной цикл** - Непрерывный цикл диалога для бесшовного взаимодействия
- **Система конфигурации** - Централизованный конфиг для простой кастомизации
- **Выбор модели** - Выбирайте предпочитаемую AI модель через конфигурацию
- **Список моделей** - Просматривайте доступные модели
- **Выполнение инструментов** - Выполняйте инструменты и команды напрямую из агента

### Запланированные возможности

#### Model Context Protocol (MCP)

- Ленивая загрузка MCP
- Интеграция с Yandex MCP

#### Улучшенный интерфейс

- Система навыков (skills)
- Markdown блоки для лучшего форматирования вывода

#### Делегирование агента

- Режимы делегирования с новым контекстом / сохранением контекста
- Оценка делегирования
- Поддержка асинхронного делегирования

#### Управление сессиями

- Режим одноразового запуска
- Сохранение и продолжение сессий
- Экспорт сессий
- Сжатие сессий

#### Управление контекстом

- Фреймирование сессии (навигация по дереву, вверх/назад)
- Скользящее окно контекста с закрепленными элементами
- Сжатие контекста
- Выборочная очистка контекста

#### Продвинутые возможности

- Подсветка шагов
- Защитные механизмы использования инструментов
- Инструментарий TODO и делегирование
- Жесткие TODO для делегирования или запуска
- Автоматическое обновление навыков
- Подсветка потребления токенов
- Самохостируемые агенты (демоны, очереди, API)

#### Интеграции

- Поддержка Figma MCP
- Доступ к файлам через SSH
- Интеграция с tmux
- Автоматизация Git/Arc

#### Инструменты разработчика

- Управление историей с поддержкой отката
- Поиск по истории чата с помощью агента
- Лимиты использования (на шаг, инструменты, делегирование)
- Абстракция файлов для под-агентов
- Исследование рассуждений и эксперименты

## Цели проекта

Это личное исследование создания минималистичного AI ассистента для программирования с приоритетом на:

- CLI-first дизайн
- Контроль и прозрачность для разработчика
- Гибкая конфигурация
- Эксперименты с архитектурами агентов
