# User Info Skill

A simple skill that stores and recalls the user's name and age. Designed to teach the agent persistent personal context so it can respond naturally when asked about the user's identity.

## When to Apply

Reference these guidelines when:

- The user asks "What is my name?" or "How old am I?"
- The agent needs to personalize responses using the user's identity
- Building context-aware conversational agents
- Testing whether the agent remembers user-provided information
- Onboarding new users or updating profile data
